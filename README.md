Internship Result Management System

A full-stack web application developed as part of a university database and interfaces coursework project. The system was designed to streamline the management and evaluation of students undertaking industrial internships by replacing manual spreadsheet-based workflows with a centralized web platform.

The application supports role-based access control, allowing administrators to manage student records, internship information, assessor assignments, and user accounts, while assessors can evaluate assigned students using predefined assessment criteria. The system automatically calculates final internship scores based on faculty-defined weightages and stores assessment feedback in a structured relational database.

Key Features

* Secure user authentication and role-based authorization
* Student profile management (create, update, delete, and view records)
* Internship and assessor assignment management
* Internship assessment and scoring system
* Automated final mark calculation using fixed weighted criteria
* Assessment feedback and comment submission
* Result viewing and management for authorized users
* Relational database design with primary and foreign key constraints
* Form validation and database-integrated CRUD operations

Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript

Learning Outcomes

This project provided practical experience in database design, SQL development, full-stack web application development, authentication systems, CRUD operations, client-server interaction, and software development within a team environment.
